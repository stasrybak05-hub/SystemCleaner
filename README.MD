# System Cleaner 🚀

Professional system optimization and cleaning tool for Windows.

---

## 🇬🇧 English
**Features:**
* **System Cleaning**: Removes temporary files, browser cache, and system logs.
* **Disk Analysis**: Finds large files (>100MB) to free up space.
* **Startup Manager**: Manage applications that run on Windows startup.
* **Monitoring**: Real-time system resource usage tracking.
* **Lightweight**: Optimized footprint (~70MB), no complex installations required.

---

## 🇷🇺 Русский
**Возможности:**
* **Очистка системы**: Удаление временных файлов, кэша браузеров и системных логов.
* **Анализ диска**: Поиск больших файлов (>100 МБ) для освобождения места.
* **Автозагрузка**: Удобное управление программами при старте Windows.
* **Мониторинг**: Отслеживание нагрузки на ресурсы ПК в реальном времени.
* **Легкость**: Оптимизированный размер (~70 МБ), не требует сложной установки.
